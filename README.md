# ASHA Whisper Project

A multilingual speech-to-text and translation backend for ASHA health worker voice notes.

---

## 🚀 Features

- 🎤 Audio transcription (Hindi / Marathi / English)
- 🌐 English translation
- 🧠 Whisper-based speech recognition
- 🔇 Silence removal (FFmpeg)
- ⚡ FastAPI backend
- 📊 Structured API response

---

## 📦 Requirements

- Python 3.10+
- FFmpeg installed
- Internet (first-time model download)
